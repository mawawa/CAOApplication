History Of Computing

First Generation Computers : Vacuum Tubes

This computer is made of vacuum tubes, for digital logic and also memory. This computer was developed under the Institute of Advanced Study IAS by John Van Newman. The computer consists of the following components.

  1. Main Memory
  2. Central processing unit
  3. Input output
  4. System bus

The components of an IAS computer system

  1. Main Memory
    this component stores data and instruction in 4096 storage locations (called words)
  2. Arithmetic Logic Unit(CA)
    This component is capable of performing operations on binary data
  3. Control Unit(CC) This component interprets the instructions in memory and causes them to be executed by fetching the from memory and executing them.

Registers

Both the Control Unit and the Arithmetic Login Unit contain registers defined a follows.

  1. Memory Buffer Register(MBR)
    This buffer contains a word that is to be stored in memory or is to be sent to the IO unit.
  2. Memory Address Buffer
    This buffer contains the address in memory of the word that is to be written into or read into the Memory Buffer Register.
  3. Instruction Register (IR)
    This contains the 8 bit opcode instruction being executed.
  4. Instruction buffer Register (IBR)
    This register holds the right hand instruction from a world in memory.
  5. Program counter (PC)
    This contains the address of the next instruction pair to be fetched from memory .
  6. Accumulator(AC) and Multiplexer Quotient (MQ) Holds the operands temporarily and the results from the ALU

The Second Generation Computers : Transistors

The transistor, which is smaller, cheaper & generates less heat replaced the vacuum tube but used the same way to construct computers
• Unlike the vacuum tube, which requires wires, metal plates, a glass capsule, and a vacuum, the transistor is a solid-state device, made from silicon. The second generation saw the introduction of more complex arithmetic and logic units and control units, the use of high- level programming languages, and the provision of system software with the computer • The size of main memory, in multiples of 210 36-bit words, grew from 2k (1k = 210) to 32k words, while the time to access one word of memory, the Memory Cycle Time, fell from 30 microseconds to 1.4 microseconds.

The Third Generation Computers: Integrated Circuits

A single, self-contained transistor is called a discrete component. • In the1950s and early 1960s, electronic equipment was composed largely of discrete components - transistors, resistors, capacitors, and so on. • These components were manufactured separately, packaged in their own containers, and soldered or wired together onto Masonite-like circuit boards, which were then installed in computers.
Whenever an electronic device called for a transistor, a little tube of metal containing a pinhead-sized piece of silicon had to be soldered to a circuit board. • The entire manufacturing process, from transistor to circuit board, was expensive and cumbersome because some computers would consist of about 10000 transistors
Here, a computer consists of gates, memory cells, and interconnections among these elements. • The gates and memory cells are, in turn, constructed of simple electronic components, such as transistors and capacitors • A gate is a device that implements a simple Boolean or logical function
