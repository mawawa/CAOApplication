Assembly Language, RISC, SISC and Parallel Processing

Parts of and Assembly program

The data Section
This section is used for declaring initialized data or constants. This data does not change during runtime. You can declare buffer sizes, file names and the likes. The syntax for declaring this is section.data
The bss Section
This section is used for declaring variables. the syntax for declaring this is section.bss
The Text Section
This section is used for keeping the actual code. This section should begin with global_start.the syntax for declaring this section is as follows: section.text global_start ; now unlike java this is a comment. "(-O-)' _start:
