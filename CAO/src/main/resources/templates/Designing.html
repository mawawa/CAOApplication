Designing for performance

The cost of computer systems continues to drop dramatically, while the perfomance and capacity of those systems continue to equally rise dramatically. The major driving factors include:

  1. Microprocessor speed
  2. Pipelining - this is when a processors works on multiple instructions at the same time
  3. Branch prediction- this is when the processor looks ahead for instructions likely to be processed.
  4. Superscalar execution - this is the ability to issue more that one instruction on every processor clock cycle.
  5. Speculative execution - here branch prediction is used along with data flow analysis to speculatively execute instruction adread of their actual appearance in the program execution
