Keyboard

The keyboard is one of the input output devices that are considered important. They are able to encode information into characters or symbols.

Monitor

The monitor is also considered as output device although in some cases it may be bundled with a touch screen which would make it and input and output device, able to display graphics to the user on a form of a screen.

Networking

The ethernet for example sits as an input output able send encoded information using the TCP IP protocol.
