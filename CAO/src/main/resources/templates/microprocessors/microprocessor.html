What is a microprocessor?

This is the central processing unit all combined into one Integrated Circuit.
The microprocessor has even memory integrated into the one Silicon chip. It is programmable, such that it is able to read instructions from memory and execute the task to deliver the intended output. A microprocessor has many components that work together eg:

  • Diodes
  • Transistors
  • Registers

Steps followed by a Microprocessor

  1. Fetch
  2. Decode
  3. Execute
